# [임용영어학] Ch.5 Binding OX 40문항 — 해설 완전 정복

<span style="color:#3182ce">**Binding**</span>은 anaphor·pronoun·R-expression이 문장 안에서 어떤 선행사를 가질 수 있는지를 규율하는 세 원리(<span style="color:#3182ce">**Principle A**</span> · <span style="color:#3182ce">**Principle B**</span> · <span style="color:#3182ce">**Principle C**</span>)로 정리됩니다. 모든 진단은 결국 "<span style="color:#3182ce">**c-command**</span>가 성립하는가" + "어느 <span style="color:#3182ce">**binding domain**</span> 안에서 bound/free인가"라는 두 축으로 환원됩니다. 아래 40문항 해설로 한 번에 정리하세요.

---

## ① c-command / binding 기본 (Q1–Q6)

### Q1. Binding의 정의 — c-command + coindexation — ✅ O

<span style="color:#3182ce">**Binding**</span>은 단순한 동일 지시(<span style="color:#3182ce">**coindexation**</span>)만으로 성립하지 않습니다. A binds B if and only if A <span style="color:#3182ce">**c-commands**</span> B and A and B are coindexed. 선행사 A가 B를 c-command하면서 동시에 coindex돼야 binding이 성립합니다. 두 조건 중 하나라도 빠지면 그 요소는 <span style="color:#319795">free</span>입니다.

### Q2. free의 정의 — ✅ O

<span style="color:#3182ce">**free**</span> = not bound. 어떤 요소를 c-command하면서 coindex된 선행사가 없으면 그 요소는 <span style="color:#319795">free</span>입니다. bound와 free는 상보적이며, <span style="color:#3182ce">**Principle B**</span>·<span style="color:#3182ce">**Principle C**</span>는 특정 영역에서 요소가 free일 것을 요구합니다.

### Q3. c-command의 표준 정의 — ✅ O

A node <span style="color:#3182ce">**c-commands**</span> its sisters and all of their descendants. X 위로 처음 갈라지는 first branching node가 지배하는 모든 요소를 X가 c-command합니다. 직관적으로 X는 자신의 sister와 그 후손들을 c-command합니다 — <span style="color:#319795">subject DP가 VP 내부 object를 c-command하는 것</span>이 전형적 예입니다.

### Q4. *The witness' testimony exposed themselves* — possessor c-command 실패 — ❌ X

<span style="color:#c53030">**⚠️ 핵심 함정**</span>. 소유격 specifier <em>the witness'</em>는 큰 subject DP 내부에 갇혀 있어 그 위 first branching node가 <em>themselves</em>를 dominate하지 못합니다 → <em>themselves</em>를 <span style="color:#c53030">c-command하지 못함</span> → 적법한 선행사 부재. <span style="color:#c53030">*The witness' testimony exposed themselves.*</span> is ungrammatical because <span style="color:#3182ce">**Principle A**</span> is violated — the anaphor has no c-commanding antecedent within its governing category. 정문은 <span style="color:#319795">*The witnesses exposed themselves.*</span>

### Q5. *The witnesses exposed themselves* — subject c-command — ✅ O

subject DP <em>the witnesses</em> 위 first branching node(TP)가 VP와 그 안의 <em>themselves</em>를 dominate하므로 subject가 reflexive를 <span style="color:#3182ce">**c-command**</span>합니다. 같은 절(=governing category) 안에서 bound되므로 <span style="color:#3182ce">**Principle A**</span>를 충족합니다 → 정문.

### Q6. coindexation만으로 binding 성립? — ❌ X

<span style="color:#c53030">**⚠️ 오답 포인트**</span>. coindexation만으로는 부족합니다 — <span style="color:#3182ce">**c-command**</span>도 반드시 필요합니다. 같은 index를 공유해도 선행사가 그 요소를 c-command하지 못하면(예: 소유격 안 선행사) binding이 아닙니다. 그래서 c-command 없는 coindexation은 <span style="color:#3182ce">**binding**</span>으로 치지 않습니다.

## ② Principle A — anaphors (Q7–Q16)

### Q7. *The actress blamed himself* — gender agreement 위반 — ❌ X

<span style="color:#c53030">**⚠️ 함정**</span>. reflexive는 선행사와 person·number·gender에서 일치해야 합니다. <em>the actress</em>(여성)와 <em>himself</em>(남성)는 <span style="color:#c53030">gender가 어긋나므로</span> 적법한 c-commanding antecedent가 없는 것과 같습니다. <span style="color:#c53030">*The actress blamed himself.*</span> is ungrammatical because <span style="color:#3182ce">**Principle A**</span> is violated — the anaphor is not properly bound in its governing category.

### Q8. *Brian hopes that Karen will forgive himself* — Principle A 위반 — ✅ O

<em>himself</em>의 <span style="color:#3182ce">**governing category**</span>는 종속절(<em>Karen will forgive himself</em>)이고 그 안에는 적법한 선행사가 없습니다. 선행사 <em>Brian</em>은 절 밖에 있어 너무 멀리 있습니다. <span style="color:#c53030">*Brian hopes that Karen will forgive himself.*</span> is ungrammatical because <span style="color:#3182ce">**Principle A**</span> is violated — the anaphor is not bound within its governing category.

### Q9. *The juror distrusts each other* — reciprocal number 위반 — ❌ X

<span style="color:#c53030">**⚠️ 오답 포인트**</span>. reciprocal <em>each other</em>는 의미상 복수 선행사를 필요로 합니다. 단수 <em>the juror</em>는 이를 만족시키지 못합니다. <span style="color:#c53030">*The juror distrusts each other.*</span> is ungrammatical because <span style="color:#3182ce">**Principle A**</span> is violated — the anaphor lacks an appropriate plural antecedent in its governing category.

### Q10. *Each other resigned* — anaphor에 선행사 부재 — ❌ X

anaphor은 독립 지시를 가질 수 없어 반드시 c-commanding 선행사가 필요합니다. 주어 자리의 <em>each other</em>는 선행사가 전혀 없습니다. <span style="color:#c53030">*Each other resigned.*</span> is ungrammatical because <span style="color:#3182ce">**Principle A**</span> is violated — the anaphor is not bound (it has no c-commanding antecedent).

### Q11. finite 종속절 anaphor이 상위 절 선행사 가능? — ❌ X

<span style="color:#c53030">**⚠️ 함정**</span>. <span style="color:#3182ce">**finite**</span> 종속절은 그 자체로 anaphor의 <span style="color:#3182ce">**governing category**</span>를 형성하므로 anaphor은 그 절 안에서 bound돼야 합니다. 상위 절의 선행사는 너무 멀어 <span style="color:#3182ce">**Principle A**</span>를 위반합니다. (ECM 비정형 보문에서는 domain이 확장되지만, finite 절에서는 닫혀 있습니다.)

### Q12. *Nora believes herself to be qualified* — ECM domain 확장 — ✅ O

<span style="color:#3182ce">**ECM**</span> 보문(<em>believe X to VP</em>)에서는 하위 subject 자리에 적법한 subject-binder가 없어 anaphor의 <span style="color:#3182ce">**governing category**</span>가 matrix subject까지 확장됩니다. 따라서 <em>Nora</em>가 <em>herself</em>를 같은 domain 안에서 c-command·bind → <span style="color:#3182ce">**Principle A**</span> 충족 → <span style="color:#319795">*Nora believes herself to be qualified.*</span> 정문. 반면 <span style="color:#c53030">*Nora believes her to be qualified.*</span> (her=Nora)는 비문입니다.

### Q13. *The committee elected herself* — agreement + 선행사 부재 — ❌ X

<span style="color:#c53030">**⚠️ 오답 포인트**</span>. <em>the committee</em>는 <em>herself</em>를 c-command하지만 의미·자질 면에서 적법한 선행사가 되지 못하고, anaphor은 governing category 안에서 bound될 적절한 선행사를 찾지 못합니다. <span style="color:#c53030">*The committee elected herself.*</span> is ungrammatical because <span style="color:#3182ce">**Principle A**</span> is violated — the anaphor is not properly bound in its governing category.

### Q14. reflexive는 선행사와 gender 달라도 되나? — ❌ X

anaphor은 독립 지시가 없으므로 선행사와 자질을 공유해야 합니다. local binding이 성립해도 gender가 어긋나면 비문입니다. <span style="color:#319795">*The actress blamed herself.*</span>는 정문이지만 <span style="color:#c53030">*The actress blamed himself.*</span>는 gender 불일치로 비문입니다 — reflexive는 선행사와 person·number·gender에서 모두 <span style="color:#3182ce">**일치(agreement)**</span>해야 합니다.

### Q15. *The siblings admire each other* — reciprocal 의미 — ✅ O

reflexive와 reciprocal은 둘 다 anaphor로 <span style="color:#3182ce">**Principle A**</span>의 적용을 똑같이 받지만 의미가 다릅니다. <span style="color:#319795">*The siblings admire each other.*</span>의 reciprocal <em>each other</em>는 '서로'(each admires the other(s))를 뜻하고, reflexive <em>themselves</em>('각자 자기 자신')와 의미가 다릅니다. <em>They admire themselves</em> = each admires self / <em>They admire each other</em> = each admires the other(s).

### Q16. *Daniel said that the teacher praised himself* — himself=Daniel 가능? — ❌ X

불가합니다. <em>himself</em>의 <span style="color:#3182ce">**governing category**</span>는 finite 종속절(<em>the teacher praised himself</em>)입니다. 그 안의 subject <em>the teacher</em>가 유일한 적법 선행사이고, 절 밖의 <em>Daniel</em>로 해석하는 독해는 <span style="color:#3182ce">**Principle A**</span> 위반이라 불가합니다. (문장 자체는 <em>himself</em>=<em>the teacher</em>로 <span style="color:#319795">정문</span>입니다.)

## ③ Principle B — pronouns (Q17–Q26)

### Q17. *Megan criticized her* (her=Megan) — Principle B 위반 — ✅ O

<span style="color:#3182ce">**pronoun**</span>은 <span style="color:#3182ce">**governing category**</span> 안에서 <span style="color:#319795">free</span>여야 합니다. <em>her</em>가 같은 절 subject <em>Megan</em>에 bound되면(her=Megan) 위반입니다. <span style="color:#c53030">*Megan criticized her.*</span> is ungrammatical on that reading because <span style="color:#3182ce">**Principle B**</span> is violated — the pronoun is not free in its governing category.

### Q18. *Megan said that Paula admired her* (her=Megan) — non-local coreference — ✅ O

<em>her</em>의 <span style="color:#3182ce">**governing category**</span>는 종속절(<em>Paula admired her</em>)이고 그 안에서는 <span style="color:#319795">free</span>입니다(Paula에 bound 안 됨). 절 밖의 <em>Megan</em>과 coreference는 <span style="color:#3182ce">**Principle B**</span>를 위반하지 않으므로 her=Megan 독해가 가능합니다.

### Q19. pronoun이 governing category 안에서 bound돼야 한다? — ❌ X

<span style="color:#c53030">**⚠️ 핵심 함정**</span>. governing category 안에서 bound돼야 하는 것은 anaphor(<span style="color:#3182ce">**Principle A**</span>)입니다. pronoun은 정반대로 같은 domain 안에서 <span style="color:#319795">free</span>여야 하고(<span style="color:#3182ce">**Principle B**</span>), domain 밖 선행사와의 coreference만 허용됩니다. 그래서 reflexive와 pronoun은 <span style="color:#3182ce">**complementary distribution**</span>을 이룹니다.

### Q20. *The senator believes him to be honest* (him=the senator) — ECM pronoun bound — ❌ X

<span style="color:#c53030">**⚠️ 오답 포인트**</span>. <span style="color:#3182ce">**ECM**</span> 보문에서 pronoun <em>him</em>의 <span style="color:#3182ce">**governing category**</span>는 matrix까지 확장되어 <em>the senator</em>를 포함합니다. him=the senator로 해석하면 같은 domain 안에서 bound되는 것입니다. <span style="color:#c53030">*The senator believes him to be honest.*</span> is ungrammatical because <span style="color:#3182ce">**Principle B**</span> is violated — the pronoun is not free in its governing category.

### Q21. *Oliver wants Sarah to support him* (him=Oliver) — non-local coreference — ✅ O

<em>him</em>의 <span style="color:#3182ce">**governing category**</span> 안에서는 <em>Sarah</em>에 bound되지 않아 <span style="color:#319795">free</span>입니다. matrix subject <em>Oliver</em>는 그 domain 밖에 있으므로 him=Oliver coreference가 <span style="color:#3182ce">**Principle B**</span>를 위반하지 않습니다 → 가능.

### Q22. reflexive와 pronoun은 complementary distribution — ✅ O

<span style="color:#3182ce">**Principle A**</span>는 anaphor이 governing category 안에서 bound될 것을, <span style="color:#3182ce">**Principle B**</span>는 pronoun이 같은 domain에서 free일 것을 요구합니다. 따라서 한 자리에서 한쪽이 정문이면 다른 쪽은 비문이 되어 <span style="color:#3182ce">**complementary distribution**</span>을 이룹니다: <span style="color:#319795">*Victor defended himself.*</span> ✓ / <span style="color:#c53030">*Victor defended him.*</span> (him=Victor) ✗.

### Q23. *Victor defended him* (him=Victor) — pronoun locally bound — ❌ X

pronoun은 같은 절(governing category) 안에서 <span style="color:#319795">free</span>여야 합니다. him=Victor로 해석하면 같은 절 subject에 bound되는 것입니다. <span style="color:#c53030">*Victor defended him.*</span> is ungrammatical on that reading because <span style="color:#3182ce">**Principle B**</span> is violated — the pronoun is not free in its governing category.

### Q24. *Victor defended himself* (himself=Victor) — anaphor locally bound — ✅ O

<em>himself</em>는 같은 절 subject <em>Victor</em>에 c-command·coindex되어 <span style="color:#3182ce">**governing category**</span> 안에서 bound됩니다. anaphor은 local binding을 요구하므로 <span style="color:#3182ce">**Principle A**</span>를 충족하는 <span style="color:#319795">정문</span>입니다.

### Q25. pronoun은 문장 안 어디서도 선행사를 못 가진다? — ❌ X

<span style="color:#c53030">**⚠️ 함정**</span>. <span style="color:#3182ce">**Principle B**</span>는 governing category 안에서만 free를 요구합니다. domain 밖 선행사와의 coreference는 허용되므로(예: <span style="color:#319795">*Oliver wants Sarah to support him.*</span>, him=Oliver) pronoun이 문장 안에서 선행사를 가질 수 있습니다.

### Q26. *Claire told Beth about her* — her=Claire 가능? — ❌ X

불가합니다. <em>her</em>의 <span style="color:#3182ce">**governing category**</span>는 그 문장 전체(단일 절)이고, subject <em>Claire</em>가 <em>her</em>를 c-command합니다. her=Claire로 해석하면 같은 domain 안에서 bound되는 것입니다. <span style="color:#c53030">*Claire told Beth about her.*</span> (her=Claire) — the reading is blocked because <span style="color:#3182ce">**Principle B**</span> is violated — the pronoun is not free in its governing category.

## ④ Principle C — R-expressions (Q27–Q33)

### Q27. R-expression은 binding domain 밖에서는 bound 가능? — ❌ X

<span style="color:#c53030">**⚠️ 핵심 함정**</span>. domain 밖 선행사와의 binding을 허용하는 것은 pronoun(<span style="color:#3182ce">**Principle B**</span>)입니다. <span style="color:#3182ce">**R-expression**</span>은 스스로 지시를 가지므로 어떤 c-commanding coindexed 선행사에도 bound될 수 없습니다 — governing category 안이든 밖이든 전부. <span style="color:#3182ce">**Principle C**</span>는 R-expression이 <span style="color:#319795">everywhere free</span>일 것을 요구합니다.

### Q28. *He thinks Trevor will win* (He=Trevor) — Principle C 위반 — ✅ O

matrix subject <em>He</em>는 종속절 안 <em>Trevor</em>를 <span style="color:#3182ce">**c-command**</span>합니다. He=Trevor로 coindex하면 R-expression이 c-commanding 선행사에 bound되는 것입니다. <span style="color:#c53030">*He thinks Trevor will win.*</span> (He=Trevor) is ungrammatical because <span style="color:#3182ce">**Principle C**</span> is violated — the R-expression is not free.

### Q29. *His mother adores Trevor* (His=Trevor) — possessor c-command 못 함 — ✅ O

<em>His</em>는 subject DP <em>His mother</em> 안의 소유격 specifier에 갇혀 <em>Trevor</em>를 <span style="color:#c53030">c-command하지 못합니다</span>. c-command가 없으면 coindex해도 binding이 아니므로 R-expression <em>Trevor</em>는 <span style="color:#319795">free</span>로 유지되어 <span style="color:#319795">*His mother adores Trevor.*</span> (His=Trevor) 독해가 가능합니다.

### Q30. *Sheila praised Sheila* (same person) — 이름 반복 bound — ❌ X

앞의 subject <em>Sheila</em>가 object 자리의 이름 <em>Sheila</em>를 <span style="color:#3182ce">**c-command**</span>합니다. 같은 인물로 coindex하면 R-expression이 bound되는 것입니다. <span style="color:#c53030">*Sheila praised Sheila.*</span> (same person) is ungrammatical on that reading because <span style="color:#3182ce">**Principle C**</span> is violated — the R-expression is not free.

### Q31. *When she arrived, Diana smiled* (she=Diana) — 부사절 c-command 없음 — ✅ O

<em>she</em>는 부사절(<em>When she arrived</em>) 안에 있어 주절 subject <em>Diana</em>를 <span style="color:#c53030">c-command하지 못합니다</span>. c-command가 없으면 coindex해도 binding이 아니므로 R-expression <em>Diana</em>가 <span style="color:#319795">free</span>로 유지되어 <span style="color:#319795">*When she arrived, Diana smiled.*</span> (she=Diana) 독해가 가능합니다.

### Q32. *She said that Diana was promoted* (She=Diana) — matrix pronoun이 이름 c-command — ❌ X

matrix subject <em>She</em>는 종속절 안 <em>Diana</em>를 <span style="color:#3182ce">**c-command**</span>합니다. She=Diana로 coindex하면 R-expression이 bound됩니다. <span style="color:#c53030">*She said that Diana was promoted.*</span> (She=Diana) is ungrammatical because <span style="color:#3182ce">**Principle C**</span> is violated — the R-expression is not free.

### Q33. 다른 절이면 c-commanding pronoun이 이름을 bound 가능? — ❌ X

<span style="color:#c53030">**⚠️ 함정**</span>. <span style="color:#3182ce">**Principle C**</span>는 R-expression이 <em>어디서나</em> <span style="color:#319795">free</span>일 것을 요구합니다. 다른 절에 있는 c-commanding pronoun이라도 coindex하면 위반입니다(예: <span style="color:#c53030">*She said that Diana was promoted.*</span>, She=Diana). locality는 R-expression에 완화를 주지 않습니다.

## ⑤ governing category / binding domain / locality (Q34–Q37)

### Q34. binding domain의 정의 — ✅ O

<span style="color:#3182ce">**binding domain**</span>(= <span style="color:#3182ce">**governing category**</span>)은 그 요소와 subject를 포함하는 가장 작은 절입니다. locality는 <span style="color:#3182ce">**Principle A**</span>·<span style="color:#3182ce">**Principle B**</span>가 적용되는 '어디까지'를 정합니다. anaphor은 이 안에서 bound, pronoun은 이 안에서 free여야 합니다.

### Q35. reflexive와 선행사는 clause-bounded — ✅ O

anaphor은 자신의 <span style="color:#3182ce">**governing category**</span> 안에서 bound돼야 하므로 선행사와 <span style="color:#3182ce">**clause-mate**</span>여야 합니다(finite 절 기준). 선행사가 다른 절에 있으면 <span style="color:#3182ce">**Principle A**</span> 위반입니다. (ECM 비정형 보문에서는 domain이 위로 확장됩니다.)

### Q36. ECM에서 matrix subject가 X의 binding domain 안 — ✅ O

비정형/<span style="color:#3182ce">**ECM**</span> 보문(<em>believe X to be…</em>)에서는 하위 subject 자리에 적법한 subject-binder가 없어 X의 <span style="color:#3182ce">**binding domain**</span>이 matrix subject까지 확장됩니다. 그래서 <span style="color:#319795">*Nora believes herself to be…*</span>는 정문(anaphor bound), <span style="color:#c53030">*Nora believes her to be…*</span>는 비문(pronoun bound)입니다.

### Q37. governing category와 binding domain은 충돌하는 다른 개념? — ❌ X

<span style="color:#c53030">**⚠️ 함정**</span>. 둘은 같은 개념의 표준 <span style="color:#3182ce">**동의어(synonyms)**</span>입니다. <em>governing category</em>는 GB 고전 용어, <em>binding domain</em>은 같은 개념의 운영 용어로, 둘 다 '요소와 그 binder가 될 수 있는 subject를 포함하는 가장 작은 절'을 가리킵니다. 충돌하는 개념이 아니라 같은 것을 부르는 두 이름입니다.

---

## ⑥ 통합 / 대조 / 기출 각도 (Q38–Q40)

### Q38. *Helen convinced herself* (✓) vs *Helen convinced her* (✗) — A vs B complementarity — ✅ O

같은 <span style="color:#3182ce">**governing category**</span> 안에서 anaphor <em>herself</em>는 bound돼야 정문이고(<span style="color:#3182ce">**Principle A**</span>), pronoun <em>her</em>는 free여야 하므로 같은 선행사로는 비문입니다(<span style="color:#3182ce">**Principle B**</span>): <span style="color:#319795">*Helen convinced herself.*</span> ✓ vs <span style="color:#c53030">*Helen convinced her.*</span> (her=Helen) ✗. 이 대조가 두 원리의 <span style="color:#3182ce">**complementarity**</span>를 보여줍니다.

### Q39. *Nora believes herself to be qualified*에서 herself를 her로 바꿔도 정문? — ❌ X

<span style="color:#c53030">**⚠️ 핵심 함정**</span>. <span style="color:#3182ce">**ECM**</span> 구문에서 <span style="color:#3182ce">**binding domain**</span>이 matrix까지 확장되므로, pronoun <em>her</em>가 그 domain 안 matrix subject <em>Nora</em>에 bound되면 위반입니다. <span style="color:#c53030">*Nora believes her to be qualified.*</span> (her=Nora) is ungrammatical on that reading because <span style="color:#3182ce">**Principle B**</span> is violated — the pronoun is not free in its governing category. 같은 자리에서 anaphor <em>herself</em>는 <span style="color:#319795">정문</span>, pronoun <em>her</em>는 비문이라 <span style="color:#3182ce">**complementarity**</span>가 ECM 환경에서도 성립합니다.

### Q40. anaphor의 유일한 선행사가 먼 matrix subject뿐이면 ill-formed — ✅ O

anaphor은 자신의 <span style="color:#3182ce">**governing category**</span>(minimal clause) 안에서 bound돼야 하므로, 유일한 선행사가 멀리 떨어진 matrix subject뿐이면 binding condition을 만족시킬 수 없어 ill-formed입니다. <span style="color:#dd6b20">**현직쌤 팁**: 기출에서는 Theta+Binding+Movement 결합형 서술형으로 출제되니, anaphor을 보면 항상 "이 binding domain 안에 적법한 c-commanding 선행사가 있나?"부터 물어라.</span>
