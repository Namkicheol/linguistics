# [임용영어학] Ch.6 Morphology OX 28문항 — 해설 완전 정복

<span style="color:#3182ce;">**Morphology**</span>는 결국 두 개의 축으로 정리됩니다 — 단어를 쪼개는 단위(<span style="color:#3182ce;">**morpheme**</span>: free/bound, <span style="color:#3182ce;">**inflection**</span> vs <span style="color:#3182ce;">**derivation**</span>)와, 새 단어를 만드는 방법(<span style="color:#3182ce;">**word-formation**</span>: conversion·blending·backformation·compounding 등). 여기에 한 morpheme이 환경 따라 다른 소리로 실현되는 <span style="color:#3182ce;">**allomorphy**</span>와 복합어의 <span style="color:#3182ce;">**headedness**</span>가 더해집니다. 아래 28문항 해설로 한 번에 정리하세요.

---

## ① Morpheme — free / bound (Q1–Q3)

### Q1. *bound morpheme*은 단독으로 설 수 없다 — ✅ O

A <span style="color:#3182ce;">**bound morpheme**</span> cannot stand alone as a word and must attach to another form, as with *-ness* in *brightness* or *re-* in *reopen*. bound morpheme은 혼자 단어가 되지 못하고 반드시 base에 부착됩니다. *-ness*·*re-*처럼 모든 <span style="color:#3182ce;">**affix**</span>가 여기에 속합니다. 반대로 <span style="color:#319795;">*bright*·*open*</span>은 단독으로 설 수 있는 <span style="color:#3182ce;">**free morpheme**</span>입니다.

### Q2. 영어의 모든 affix는 bound morpheme이다 — ✅ O

In English, every affix — both <span style="color:#3182ce;">**prefixes**</span> and <span style="color:#3182ce;">**suffixes**</span> — is a bound morpheme. 영어에서 affix는 prefix이든 suffix이든 단독으로 단어가 될 수 없어 항상 base에 붙어야 합니다. 따라서 모든 affix는 정의상 <span style="color:#319795;">bound morpheme</span>입니다.

### Q3. morpheme = 단독으로 설 수 있는 최소 단위? — ❌ X

<span style="color:#c53030;">**⚠️ 정의 함정**</span>. A morpheme is best defined as <span style="color:#c53030;">the smallest unit that can *always* stand alone as an independent word</span> — 이 진술은 틀렸습니다. morpheme은 <span style="color:#3182ce;">**minimal unit of meaning or grammatical function**</span>으로 정의됩니다. 단독으로 설 수 있는 것은 <span style="color:#319795;">free morpheme</span>에 한정되며, *-ed*·*un-* 같은 bound morpheme은 독립 단어가 될 수 없습니다. '항상 단독으로 선다'는 free morpheme에만 해당하는 성질입니다.

## ② Inflection vs Derivation (Q4–Q9)

### Q4. 영어 inflection은 정확히 8개, 모두 suffix — ✅ O

English has exactly <span style="color:#3182ce;">**eight inflectional morphemes**</span>, and all of them are suffixes. 영어의 8 inflections는 noun *-'s*(possessive)·*-s*(plural), verb *-s*(3sg)·*-ing*·*-ed*·*-en*, adjective *-er*(comparative)·*-est*(superlative)이며, 모두 <span style="color:#319795;">suffix</span>입니다. <span style="color:#3182ce;">**inflectional prefix**</span>는 영어에 존재하지 않습니다.

### Q5. inflection은 품사를 절대 바꾸지 않는다 — ✅ O

An <span style="color:#3182ce;">**inflectional morpheme**</span> never changes the grammatical category of the word it attaches to. inflection은 수·시제·비교·소유 같은 문법 기능만 표시하고 품사를 유지합니다 — <span style="color:#3182ce;">**category-preserving**</span>. <span style="color:#319795;">*walk → walks/walked/walking*</span>은 모두 verb로 남고, *tall → taller/tallest*는 모두 adjective입니다.

### Q6. *-ize*는 class-changing derivational suffix — ✅ O

The suffix *-ize* in *standardize* is a <span style="color:#3182ce;">**class-changing**</span> derivational affix. *standard*(N/A) + *-ize* → <span style="color:#319795;">*standardize*(V)</span>로 품사가 바뀝니다. 새 단어를 만들고 품사를 바꾸는 affix는 <span style="color:#3182ce;">**derivational**</span>이며, 그중 품사를 바꾸는 것을 class-changing이라 합니다.

### Q7. *painter*의 *-er*는 *shorter*의 *-er*와 같은 inflection? — ❌ X

<span style="color:#c53030;">**⚠️ 동음이의 함정**</span>. The *-er* in *painter* is an <span style="color:#c53030;">inflectional morpheme, just like the *-er* in *shorter*</span> — 틀린 진술입니다. *paint*(V) + agentive *-er* → *painter*(N)는 품사를 바꾸는 <span style="color:#319795;">derivational</span>(V→N)입니다. 반면 *short → shorter*의 *-er*는 <span style="color:#319795;">comparative inflection</span>(adj→adj)으로 품사를 유지합니다. 형태는 같아도 서로 다른 morpheme입니다.

> **비교** — derivational *-er*: *painter* (V→N), agentive / inflectional *-er*: *shorter* (adj), comparative

### Q8. *unfriendly*의 *un-*는 inflectional prefix? — ❌ X

<span style="color:#c53030;">**⚠️ 함정**</span>. The *un-* in *unfriendly* is an <span style="color:#c53030;">inflectional prefix</span> — 틀렸습니다. 영어의 inflection 8개는 모두 suffix이므로 inflectional prefix는 존재하지 않습니다. *un-*는 새 단어를 만드는 <span style="color:#319795;">derivational prefix</span>(adj→adj)입니다.

### Q9. 영어 inflection은 12개이고 일부는 prefix? — ❌ X

<span style="color:#c53030;">**⚠️ 개수 함정**</span>. English has <span style="color:#c53030;">twelve inflectional morphemes, several of which are prefixes</span> — 두 군데 다 틀린 진술입니다. 영어의 inflection은 정확히 <span style="color:#319795;">8개</span>이고 전부 <span style="color:#319795;">suffix</span>입니다. '12개'·'prefix 포함'은 모두 오답 포인트입니다.

## ③ Word-formation processes (Q10–Q16)

### Q10. *They will salt the soup* — conversion (zero-derivation) — ✅ O

In *They will salt the soup*, *salt* is formed by <span style="color:#3182ce;">**conversion (zero-derivation)**</span>, with no affix added. 명사 *salt*를 동사로 그대로 쓰는 것은 affix를 더하지 않고 품사만 바꾸는 conversion(zero-derivation, functional shift)입니다. <span style="color:#319795;">형태 변화 없이 syntactic category만 바뀝니다</span>(N→V).

### Q11. *smog* (smoke + fog) — blending — ✅ O

*Smog*, from *smoke* + *fog*, is formed by <span style="color:#3182ce;">**blending**</span>. blending은 두 단어의 <span style="color:#3182ce;">**non-morphemic**</span> 조각(보통 앞 단어 앞부분 + 뒤 단어 뒷부분)을 결합합니다. *smoke*의 앞부분과 *fog*의 뒷부분이 합쳐져 <span style="color:#319795;">*smog*</span>가 됩니다.

### Q12. *edit* ← *editor* — backformation — ✅ O

*Edit* was created from *editor* by <span style="color:#3182ce;">**backformation**</span>. backformation은 실제 또는 추정 affix를 제거해 새 단어를 만드는 과정입니다. *editor*를 '*edit* + 행위자 *-or*'로 재분석해 동사 <span style="color:#319795;">*edit*</span>을 만들어낸 것이 그 예입니다.

### Q13. *brunch* (breakfast + lunch) — clipping? — ❌ X

<span style="color:#c53030;">**⚠️ clipping vs blending 함정**</span>. *Brunch* is formed by <span style="color:#c53030;">clipping</span> — 틀렸습니다. <span style="color:#3182ce;">**clipping**</span>은 한 다음절 단어의 음절을 잘라내는 것입니다(*professor → prof*). *brunch*는 *breakfast*와 *lunch* 두 단어의 조각을 합친 것이므로 <span style="color:#319795;">blending</span>입니다.

> **비교** — Clipping: 한 단어 줄임 (*advertisement → ad*) / Blending: 두 단어 조각 (*breakfast + lunch → brunch*)

### Q14. *FBI* (글자 따로 발음) — acronym? — ❌ X

<span style="color:#c53030;">**⚠️ acronym vs initialism 함정**</span>. *FBI*, pronounced as separate letters, is an <span style="color:#c53030;">acronym</span> — 틀렸습니다. <span style="color:#3182ce;">**acronym**</span>은 구의 머리글자를 한 단어처럼 발음하는 것입니다(*NATO*, *radar*). *FBI*처럼 글자를 하나하나 따로 읽으면 <span style="color:#319795;">initialism</span>(abbreviation)입니다.

> **비교** — Acronym: *NATO*, *laser* — 단어처럼 발음 / Initialism: *FBI*, *USA* — 글자 따로 발음

### Q15. *standardize* — conversion (zero-derivation)? — ❌ X

<span style="color:#c53030;">**⚠️ conversion vs derivation 함정**</span>. *Standardize* is an example of <span style="color:#c53030;">conversion (zero-derivation)</span> — 틀렸습니다. <span style="color:#3182ce;">**conversion**</span>은 affix를 더하지 않고 품사만 바꾸는 것입니다. *standardize*는 *-ize*라는 derivational suffix가 추가되었으므로 <span style="color:#319795;">derivation</span>에 해당합니다.

### Q16. *loveseat* (love + seat) — derivation? — ❌ X

<span style="color:#c53030;">**⚠️ derivation vs compounding 함정**</span>. *Loveseat* is formed by <span style="color:#c53030;">derivation</span> — 틀렸습니다. <span style="color:#3182ce;">**derivation**</span>은 base에 derivational affix를 붙입니다. *loveseat*은 *love*와 *seat*라는 두 free word를 결합한 것이므로 <span style="color:#319795;">compounding</span>입니다.

## ④ Allomorphy (Q17–Q20)

### Q17. plural [s]/[z]/[əz] — phonologically conditioned allomorphs — ✅ O

[s], [z], and [əz] are three <span style="color:#3182ce;">**phonologically conditioned allomorphs**</span> of the English plural morpheme. plural *-s*는 무성음 뒤 [s](*maps*), 유성음 뒤 [z](*beds*), <span style="color:#3182ce;">**sibilant**</span> 뒤 [əz](*buzzes*)로 실현됩니다. <span style="color:#319795;">의미·기능은 같고 소리만 조건에 따라 달라지는</span> phonologically conditioned allomorph입니다.

### Q18. negative prefix는 bilabial 앞에서 *im-* — ✅ O

The negative prefix appears as *im-* before bilabial sounds, as in *imbalance*. negative *in-*는 후행 자음의 조음 위치에 <span style="color:#3182ce;">**place assimilation**</span>됩니다. bilabial /b, p, m/ 앞에서는 <span style="color:#319795;">*im-*</span>(*imbalance, impossible*), /l/ 앞 *il-*, /r/ 앞 *ir-*, 그 밖에서는 *in-*입니다.

> **비교** — *im- / il- / ir-*: bilabial / /l/ / /r/ 앞 / *in-*: 그 밖의 환경

### Q19. plural [əz]가 voiceless stop 뒤(*mats*)에 나타난다? — ❌ X

<span style="color:#c53030;">**⚠️ 조건 함정**</span>. The plural allomorph [əz] occurs after <span style="color:#c53030;">voiceless stops such as the /t/ in *mats*</span> — 틀렸습니다. <span style="color:#3182ce;">**[əz]**</span>는 sibilant(/s, z, ʃ, ʒ, tʃ, dʒ/) 뒤에서만 쓰입니다(*bushes*). *mats*는 무성음 /t/로 끝나므로 plural이 <span style="color:#319795;">[s]</span>로 실현됩니다.

### Q20. 한 morpheme의 두 allomorph는 의미가 서로 다르다? — ❌ X

<span style="color:#c53030;">**⚠️ 핵심 함정**</span>. Two allomorphs of one morpheme <span style="color:#c53030;">differ in meaning</span> from each other — 틀렸습니다. <span style="color:#3182ce;">**allomorph**</span>는 한 morpheme이 음운 환경에 따라 다른 형태로 실현된 것입니다. <span style="color:#319795;">의미·문법 기능은 같고 소리(형태)만 조건에 따라 달라집니다</span>. [s]/[z]/[əz]는 모두 '복수'라는 같은 기능을 합니다.

## ⑤ Compounding — headedness (Q21–Q26)

### Q21. 영어 compound의 head는 rightmost element — ✅ O

In English compounds, the <span style="color:#3182ce;">**rightmost element is the head**</span> and determines the grammatical category of the whole. <span style="color:#319795;">*handmade*(N+A→A)</span>처럼 마지막 요소의 품사가 전체 품사를 결정하고, 대략적 의미(broad meaning)도 head가 좌우합니다.

> **비교** — head = rightmost: *handmade* → A (*made*) / 의미: head가 broad meaning 결정

### Q22. *teapot* — endocentric compound — ✅ O

*Teapot* is an <span style="color:#3182ce;">**endocentric compound**</span> because it denotes a kind of pot. <span style="color:#3182ce;">**endocentric**</span> compound는 head의 <span style="color:#3182ce;">**hyponym**</span>으로, head가 가리키는 것의 한 종류를 뜻합니다. <span style="color:#319795;">*teapot*은 일종의 pot</span>이므로 endocentric입니다(*raincoat* = a kind of coat도 동일).

### Q23. *pickpocket* — exocentric compound — ✅ O

*Pickpocket* is an <span style="color:#3182ce;">**exocentric compound**</span> because it denotes a person, not a kind of pocket. <span style="color:#3182ce;">**exocentric**</span>(headless) compound는 두 요소 중 어느 것의 하위어도 아닙니다. <span style="color:#319795;">*pickpocket*은 pocket의 한 종류가 아니라 '소매치기(사람)'</span>를 가리키므로 exocentric입니다(*scarecrow*도 동일).

### Q24. *overflow*에서 head는 *over*? — ❌ X

<span style="color:#c53030;">**⚠️ head 함정**</span>. In the English compound *overflow*, <span style="color:#c53030;">*over* is the head</span> — 틀렸습니다. 영어 compound head는 항상 오른쪽 요소(rightmost)입니다. *overflow*는 *over*(preposition) + *flow*(verb)로, head는 <span style="color:#319795;">*flow*</span>이고 전체가 verb가 됩니다.

### Q25. *greenhouse*와 *green house*는 강세·의미가 동일? — ❌ X

<span style="color:#c53030;">**⚠️ compound vs phrase 함정**</span>. *Greenhouse* and *green house* have <span style="color:#c53030;">identical stress and meaning</span> — 틀렸습니다. compound는 보통 첫 요소에 강세(<span style="color:#319795;">*gréenhouse* '온실'</span>)가 오고, 대응하는 phrase는 둘째 요소에 강세(<span style="color:#319795;">*green hóuse* '초록색으로 칠한 집'</span>)가 옵니다. <span style="color:#3182ce;">**강세 위치**</span>가 둘을 구별합니다.

> **비교** — Compound: *gréenhouse* '온실' (첫 요소 강세) / Phrase: *green hóuse* '초록 집' (둘째 강세)

### Q26. *toothbrush*의 복수는 *teethbrush*? — ❌ X

<span style="color:#c53030;">**⚠️ inflection 위치 함정**</span>. The plural of *toothbrush* is <span style="color:#c53030;">teethbrush, inflecting the first element</span> — 틀렸습니다. compound의 inflection은 첫 요소가 아니라 <span style="color:#319795;">복합어 전체</span>에 부착됩니다. 따라서 복수형은 <span style="color:#c53030;">*\*teethbrush*</span>가 아니라 <span style="color:#319795;">*toothbrushes*</span>입니다.

## ⑥ Word structure — tree / bracketing (Q27–Q28)

### Q27. *unbuttonable*은 구조적 중의성을 가진다 — ✅ O

*Unbuttonable* is structurally ambiguous between <span style="color:#3182ce;">**[un-[buttonable]]**</span> ('not able to be buttoned') and <span style="color:#3182ce;">**[[unbutton]-able]**</span> ('able to be unbuttoned'). *un-*는 두 morpheme입니다: adjectival *un-*(adj→adj 'not')와 verbal *un-*(V→V 'reverse'). 전자가 *buttonable*에 붙으면 '단추를 채울 수 없는', 후자가 *button*에 먼저 붙어 *unbutton*이 되고 *-able*이 붙으면 '단추를 풀 수 있는'이 됩니다. <span style="color:#3182ce;">**bracketing ambiguity**</span>의 전형입니다.

> **비교** — [un-[buttonable]]: 'not able to be buttoned' / [[unbutton]-able]: 'able to be unbuttoned'

### Q28. *rewashable* = [re-[washable]]로 분석해야 한다? — ❌ X

<span style="color:#c53030;">**⚠️ 선택 자질에 따른 순서 함정**</span>. *Rewashable* must be analyzed as <span style="color:#c53030;">[re-[washable]]</span>, with *re-* attaching to the adjective *washable* — 틀렸습니다. <span style="color:#3182ce;">**re-**</span>는 verb를 입력으로 선택하므로 *rewash*(V)를 먼저 만들고 그 위에 *-able*이 붙습니다 → <span style="color:#319795;">**[[rewash]-able]**</span>. *washable*은 adjective라서 *re-*가 붙을 수 없어 <span style="color:#c53030;">*\*[re-[washable]]*</span>는 불가능합니다.

<span style="color:#dd6b20;">**현직쌤 팁**: morphology OX는 ① process 식별(conversion·blending·clipping·backformation·acronym vs initialism), ② inflection vs derivation 구분(특히 동음 *-er* 함정), ③ compound headedness(rightmost head, endo/exocentric, 강세·복수 위치)가 3대 빈출 축입니다. 단어를 보면 항상 "affix가 더해졌나(derivation), 아니면 그냥 품사만 바뀌었나(conversion)?", "이 *-er*는 V→N인가 adj→adj인가?", "이 복합어는 head의 한 종류인가(endocentric)?"를 먼저 물으세요.</span>
