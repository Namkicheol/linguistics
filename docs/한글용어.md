# 한글 용어 사용 규칙 — linguistics

> **`_study.html` 개념 정리, OX 퀴즈 해설(`why` 필드), 챕터 도입부 등 한글이 등장하는 모든 본문에 적용되는 용어 정책.**
> 핵심: 한글 설명은 유지하되, 그 안의 학술 키워드/개념 용어는 영어 원어로 통일한다.
> 임용 관련 모든 레포(testmaster, englisheducation, phonetics-phonology, transformational-grammar, linguistics)에 공통 적용되는 정책. **원본 정책: 전역 `~/.claude/CLAUDE.md`** + **testmaster `docs/한글용어.md`**.

---

## 원칙

1. **한글 설명 자체는 유지** — 수험생을 위한 한글 풀이/요약/맥락 설명은 그대로 둔다. 한글 블록을 통째로 삭제하지 않는다.
2. **키워드 개념 용어는 영어 원어로** — 학술 용어를 한글 번역 조어로 옮기면 어색하므로 영어 원어로 표기한다.
3. **refs(합격자노트·강사 교재 MD·기출 원문) 표현 우선** — refs에 한글 표현이 있으면 그것을 쓴다. 없으면 영어 원어.
4. **혼용 금지** — 같은 개념을 한 페이지 안에서 한글/영어 혼용 금지. 한쪽으로 통일.
5. **표준 통사 원리·이론 이름은 항상 영어** — EPP, ECP, Theta Criterion, Case Filter, Subjacency, Binding Principle A/B/C, c-command, Move α 등.

### 🥇 통사론 한글 용어 1차 출처 — 정태구 『영어통사론』

통사론(Syntax) 한글 표현이 필요할 때 다음을 우선 참조한다:

- **`refs/TalkFile_영어통사론 정태구.md`** + `refs/TalkFile_영어통사론 정태구_images/`
- 정태구 교수가 사용하는 한글 학술 표현은 임용 전공영어 통사론 표준이므로, 본 레포의 통사론 한글 용어가 정태구 자료와 충돌할 경우 **정태구 표현을 따른다**
- 정태구 자료가 PDF 이미지 형태(텍스트 OCR 미적용)이므로, 통사론 작업 중 한글 용어 결정 시 해당 페이지 이미지를 직접 확인 후 채택
- 정태구에 한글 표현이 없거나 영어 원어로만 등장 → 영어 원어 그대로 사용

## 위반 즉시 수정 트리거

- `_study.html` 본문 또는 OX 퀴즈 `why`에 위 표의 ❌ 항목이 단독 한글로 등장 → 즉시 영어 원어로 교체.
- 영어 키워드 + 한글 글로스(`c-command(C-통제)` 등) 형태에서 한글 글로스가 어색하거나 영어와 동일 의미면 → 한글 글로스 제거.
- refs 어디에도 등장하지 않는 새 한글 학술 번역어를 도입 → 즉시 영어 원어로 교체.
- 합격자 노트 / 강사 교재 / 기출 문제지 어디에도 등장하지 않는 한국어 개념어 발견 → 제거 또는 영어로 대체.
