🧩 ① Morpheme — free / bound, root / stem / base / affix 최소 의미 단위 ▼
Core Definition
A morpheme is the minimal unit of meaning or grammatical function. A free morpheme can stand alone as a word; a bound morpheme must attach to another form. All affixes in English — prefixes and suffixes — are bound morphemes.

morpheme(형태소)은 의미·문법 기능을 가지는 최소 단위다. 혼자 단어로 설 수 있으면 free, 반드시 다른 형태에 붙어야 하면 bound다.

root는 더 쪼갤 수 없는 핵심 의미 형태, stem/base는 affix가 붙는 대상이다. root는 가장 안쪽 핵심이고, stem·base는 affix가 붙는 시점의 형태라는 점을 구분한다.

affix는 bound morpheme으로, base 앞에 붙으면 prefix, 뒤에 붙으면 suffix다.

free morpheme
혼자 단어로 성립book, travel, cat, play
bound morpheme
반드시 부착 (= 모든 affix)re-, -ist, -ed, -s, -ness
unkindness = un-(prefix, bound) + kind(root, free) + -ness(suffix, bound) → 3 morphemes players = play(free root) + -er(bound, agentive) + -s(bound, plural) cats = cat(free) + -s(bound) book = single free morpheme (no internal structure) root는 free일 수도 bound일 수도 있으나, 영어에서 affix는 예외 없이 모두 bound다.

🗂️ ② Lexical vs Functional morphemes open vs closed class ▼
Core Definition
Lexical morphemes are content-bearing free morphemes (ordinary nouns, verbs, adjectives, adverbs); new ones are added easily → open class. Functional morphemes are function words (conjunctions, prepositions, articles, pronouns); new ones are almost never added → closed class.

free morpheme은 다시 lexical과 functional로 나뉜다. lexical은 메시지의 '내용'을 나르는 noun·verb·adjective 등으로, 새로 잘 만들어지므로 open class다.

functional은 conjunction·preposition·article·pronoun 같은 기능어로, 거의 추가되지 않으므로 closed class다.

보조: derivational morpheme은 새 lexical word를 만드는 데 쓰여 open class와 연결되고, inflection은 functional 성격에 가깝다.

Lexical (open class)
blog, email, download새 동사 to google 처럼 쉽게 추가
Functional (closed class)
the, of, because, them새 preposition·article은 사실상 생기지 않음

⚖️ ③ Inflection vs Derivation the core contrast ▼
Core Definition
Inflection marks grammatical function only and never changes grammatical category; English has exactly eight inflections, all suffixes. Derivation makes a new word, may change category, and uses prefixes or suffixes.

핵심 판별: 품사가 바뀌거나 새 단어가 생기면 derivation, 같은 단어의 문법형이면 inflection.

inflection은 수·시제·비교·소유 같은 문법 기능만 표시하며 새 단어를 만들지 않는다. derivation은 새 lexical word를 만들고 품사를 바꿀 수 있다.

🔧 ④ Affix types — prefix / suffix, class-changing vs class-maintaining 위치 ↔ 효과 별개 축 ▼
Core Definition
By position: prefix (before the base) vs suffix (after the base). By effect on word class: class-changing derivational affixes change part of speech; class-maintaining ones keep it.

영어 prefix는 대체로 class-maintaining(un-, re-, dis-)이고, 많은 suffix가 class-changing(-ness, -ize, -ly)이다.

position과 effect는 별개 축이다: un- 는 prefix이면서 class-maintaining, -able·-ize 는 suffix이면서 class-changing.

※ 이 축(POS 기반)은 Class 1 / Class 2 음운 분류(⑨ 참조)와는 다른 개념이다. "class"라는 단어만 공유한다.

Class-changing (category-changing)
-ness A→N: kind→kindness-ize N/A→V: modern→modernize-ful N→A: care→careful-ly A→Adv: quick→quickly
Class-maintaining
un- A→A: legal→illegalre- V→V: write→rewritedis- V→V: do→undo 류 prefix 다수
Class-changing suffix: employ(V) → employment(N); beauty(N) → beautiful(A) Class-maintaining prefix: legal(A) → illegal(A); write(V) → rewrite(V) Position vs effect 독립: un- 는 prefix이면서 class-maintaining; -ize 는 suffix이면서 class-changing.

🔀 ⑤ Allomorphy — conditioned variants of a morpheme phonologically conditioned ▼
Core Definition
An allomorph is one of the variant phonetic forms of the same morpheme (same meaning/function). The variants are typically phonologically conditioned — the surrounding sounds decide which form appears.

한 morpheme이 음운 환경에 따라 다른 형태로 실현되면 그 변이형들이 allomorph다. 의미·기능은 같고 소리만 조건에 따라 갈린다.

🏗️ ⑥ Word-formation processes derivation · compounding · conversion · clipping · blending · acronym · backformation ▼
Core Definition
새 단어를 만드는 과정들. affix를 더하면 derivation, 단어를 합치면 compounding, affix 없이 품사만 바꾸면 conversion, 잘라내면 clipping, 두 단어 조각을 섞으면 blending, 머리글자를 단어처럼 읽으면 acronym, affix를 떼어내면 backformation이다.

derivational affix 추가kind → unkindness
Compounding
둘 이상의 단어 결합love + seat → loveseat (⑦ 상세)
Conversion (zero-derivation)
affix 없이 품사 전환 (functional shift)butter(N) → to butter(V)2음절어 강세 이동: ímport(N) / impórt(V)
Clipping
한 단어의 음절 삭제professor → prof, influenza → flu
Blending
두 단어의 non-morphemic 조각 결합smoke+fog → smog, motor+hotel → motel

🔗 ⑦ Compounding — headedness, endocentric vs exocentric rightmost head · compound vs phrase ▼
Core Definition
A compound combines two or more words into a new word. In English the rightmost element is the head: it determines the compound's grammatical category and broad meaning.

endocentric compound는 head의 하위어(a kind of head)다 — armchair = a kind of chair.

exocentric compound (headless / bahuvrihi)는 어느 요소의 하위어도 아니다 — pickpocket = a person, not a pocket.

category determination: 두 부분의 품사가 다르면 오른쪽(final) 단어의 품사가 이긴다.

Endocentric (head의 하위어)
teapot = a kind of potraincoat = a kind of coatblackbird = a kind of bird
Exocentric (어느 쪽도 아님)
scarecrow = a device, not a crowturncoat = a traitor, not a coathighbrow = a person
Head 결정: overflow (preposition+verb → verb), handmade (N+A → A) Compound vs phrase (강세): bláckboard 'chalkboard' (compound, 첫 요소 강세) vs black bóard 'a board that is black' (phrase, 둘째 강세) Inflection 위치: toothbrushes (전체에 부착) teethbrush (첫 요소 inflection 불가) 복수·시제 표지는 첫 요소가 아니라 복합어 전체에 붙는다. overflow의 head는 rightmost flow이지 over가 아니다.

🌳 ⑧ Hierarchical word structure — tree / bracketing & ambiguity [un-[V-able]] vs [[un-V]-able] ▼
Core Definition
Morphologically complex words are built in ordered steps, giving hierarchical (layered) internal structure shown by a tree or bracketing. Each affix's input/output category restricts the order.

Ambiguity from structure: un- 는 두 개의 distinct morpheme이다 — (a) un- + Adjective → Adjective ('not', unhappy); (b) un- + Verb → Verb ('reverse of', untie).

그래서 un + V + able 형태는 두 가지로 bracketing되어 중의성이 생긴다. bracketing이 의미를 가른다는 점이 핵심이다.

Bracketing ambiguity: unbuttonable
Ordering fixed by selection: rewashable
Bracketing ambiguity: unbuttonable = [un-[button-able]] 'not able to be buttoned' vs [[un-button]-able] 'able to be unbuttoned' Ordering fixed: unwearable = [un-[wear-able]] only (-able 가 동사 wear에 먼저 닿아야 함; 그 뒤 adjectival un- 이 형용사 wearable에 부착) re-[washable] — washable은 adjective라 verb 전용 re- 부착 불가 각 affix의 input/output 품사 제약(selectional restriction)이 derivation 순서를 결정한다.

📈 ⑨ Productivity, lexicalization & derivational constraints 생산성 · 제약 ▼
Core Definition
Productivity is the degree to which a process applies freely to new bases. Inflection and some affixes (-ness, -ly) are highly productive; others are restricted by phonological/category constraints.

derivational constraints: 일부 affix는 base의 형태를 선택한다. -en(causative, A→V)은 monosyllabic + obstruent-final base에만 붙고, -al(V→N)은 final-syllable stress를 요구한다.

lexicalization: 파생형이 하나의 어휘로 굳어 의미가 더는 예측되지 않는 현상 (business < busy+ness, 하지만 'state of being busy'가 아님).

Productive / 적용
bright → brightenshort → shorten(monosyllabic + obstruent-final 만족)
Blocked / 차단
calmen (/m/ sonorant 종결)curiousen (disyllabic)
Level-ordering note (개념 소개 수준)

Class 1 affix는 강세 이동·분절 변화를 일으킬 수 있고(-ity, -ion), Class 2 affix는 음운적으로 중립적이다(-ness, -less). Class 1이 보통 root에 더 가깝게 위치한다.

※ 이 Class 1 / Class 2 (음운 분류)는 ④의 class-changing / class-maintaining (POS 분류)과 다른 축이다. 혼동 금지.

bright → brighten, short → shorten (productive) calmen (ends in /m/, sonorant) · curiousen (disyllabic) understand ≠ 'stand under' — lexicalized productivity는 한 과정이 새 base에 얼마나 자유롭게 적용되는가의 정도다.
